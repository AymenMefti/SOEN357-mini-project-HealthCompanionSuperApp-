# CareConnect - SOEN 357 Mini Project

**CareConnect** is a personal health companion application designed to simplify medication tracking, symptom logging, and insurance document management. This project was developed as part of the SOEN 357 (User Interface Design) course at Concordia University.

The development of CareConnect followed a user-centered design process, including:
1. **Product Research:** Competitive analysis and community feedback.
2. **User Personas:** Targeted designs for busy students and chronic patients.
3. **User Journey Mapping:** Optimizing the flow for key user scenarios.
4. **Prototyping:** Transition from low-fidelity sketches to a high-fidelity interactive Figma prototype.
